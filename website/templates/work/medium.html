Medium is a back-office solution for large art galleries and museums built to manage inventory, sales and collectors/contacts. It is capable of generating gallery-specific tearsheets, sales and offer documentation and contains robust searching for specific artworks and contacts

Mathematic designed and built the original system and then licensed in 2011.

Tooling used:

  • ExtJS
  • PHP
  • MySQL
  • Management portal built using Symfony2
  • Hosted on AWS (EC2, S3)
  • Redis for caching
  • Automated configuration management using Puppet
